# 📈 Kaggle Predictive Machine Learning Model

Bu proje, üniversite sınav gereksinimleri kapsamında Kaggle verisetleri üzerinde makine öğrenmesi modellerinin doğruluk skorunu (Accuracy Score) optimize etmek amacıyla geliştirilmiştir.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Python 3
* **Kütüphaneler:** NumPy, Pandas (Veri Manipülasyonu ve Matematiksel Modelleme)
* **Algoritmalar:** Sınıflandırma ve Regresyon Algoritmaları

## 💡 Öne Çıkan Başarılar & Optimizasyon
* Proje kapsamında ham veriler üzerinde **Özellik Mühendisliği (Feature Engineering)** uygulanarak model performansı artırılmıştır.
* **Hiperparametre Optimizasyonu** süreçleri tamamlanarak Kaggle üzerinde **%94.5** gibi yüksek bir doğruluk skoruna ulaşılmıştır.
